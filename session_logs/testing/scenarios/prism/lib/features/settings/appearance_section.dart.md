# prism/lib/features/settings/appearance_section.dart

## Unit Scenarios
- Theme mode selection updates notifier state.
- AMOLED toggle flips bool.
- Accent preset selection updates state.

## Widget Scenarios
- Theme mode chips show selected styling.
- Accent grid shows checkmark on selection.
- AMOLED toggle uses correct on/off colors.

## Integration Scenarios
- Theme change persists across app restart.
- Accent change updates navigation and buttons.
- System mode follows platform brightness.

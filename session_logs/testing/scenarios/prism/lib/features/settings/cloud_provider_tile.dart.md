# prism/lib/features/settings/cloud_provider_tile.dart

## Unit Scenarios
- Masked key displays and uses expected format.
- Test connection with empty key shows error message.
- Save persists provider config and closes tile.

## Widget Scenarios
- Expand and collapse config form.
- Get API Key and API Docs buttons open URLs.
- Test result banner shows success or failure.

## Integration Scenarios
- Configure provider -> models registered in AI service.
- Remove provider -> models removed from selector.
- OpenRouter browse models updates selection list.

# prism/lib/core/ml/ml_kit_service.dart

## Unit Scenarios
- OCR returns lines and blocks for a known image fixture.
- Entity extraction returns money and date entities.
- Smart reply returns suggestions after conversation history.
- Language ID returns a valid language code for input.

## Widget Scenarios
- Tools screen ML Kit tools show enabled states.
- OCR tool result renders recognized text.
- Entity extraction results map to finance input preview.

## Integration Scenarios
- Image pick -> OCR -> note creation flow.
- Speech input -> entity extraction -> transaction creation.
- Smart reply suggestions appear in chat footer.
